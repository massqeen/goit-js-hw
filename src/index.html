Geolocation API

API геолокации является частью объектной модели браузера - интерфейса, который предоставляет метод для работы с вкладкой.

window.navigator содержит большое количество свойств текущей вкладки браузера, в т.ч. geolocation.Это экземпляр класса Geolocation, в прототипе которого находится 3 метода:

  • getCurrentPosition
  • watchPosition - отслеживание позиции через промежутки времени, возвращает идентификатор
  • clearWatch
